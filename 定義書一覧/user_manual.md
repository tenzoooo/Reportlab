# 実験レポート自動化ツール 使用書(ユーザーマニュアル)

## 1. はじめに

この度は、「実験レポート自動化ツール」をご利用いただき、誠にありがとうございます。このツールは、AIの力を活用して、皆様の実験レポート作成にかかる時間を劇的に短縮し、より創造的で重要な「考察」に集中できるよう支援することを目的としています。

本マニュアルでは、ツールの基本的な使い方から便利な機能までを分かりやすく解説します。ぜひご一読いただき、日々の学習にお役立てください。

## 2. クイックスタートガイド

初めてお使いになる方向けに、レポート作成までの最短ステップをご紹介します。

1.  **アカウント登録**: まずは[こちら](https://example.com/register)からアカウントを登録します。
2.  **ログイン**: 登録した情報で[ログイン](https://example.com/login)します。
3.  **新規レポート作成**: ダッシュボードの「新規レポート作成」ボタンをクリックします。
4.  **データアップロード**: 実験データ(Excelファイルや写真など)をアップロードします。
5.  **生成開始**: 「レポートを生成」ボタンをクリックします。AIが分析と文書作成を開始します。
6.  **ダウンロード**: 生成が完了したら、Word形式のレポートをダウンロードします。

## 3. 主な機能と使い方

### 3.1. アカウント管理

#### アカウント登録

初めてご利用になる際は、メールアドレスとパスワードを設定してアカウントを登録してください。登録したメールアドレスに確認メールが送信されますので、メール内のリンクをクリックして登録を完了してください。

#### ログイン

登録済みのメールアドレスとパスワードでログインします。ログイン後は、過去に作成したレポートなどを管理できる個人用のダッシュボードが表示されます。

### 3.2. レポートの作成

#### ステップ1: 新規作成の開始

ダッシュボードにある「新規レポート作成」ボタンをクリックすると、レポート作成ページに移動します。

#### ステップ2: 実験データのアップロード

レポートの元となるデータをアップロードします。複数のファイルを同時にアップロードすることも可能です。

-   **Excelファイル**: 数値データや表が含まれるExcelシートをそのままアップロードできます。
-   **画像ファイル**: オシロスコープの画面を撮影した写真や、実験装置の画像(JPEG, PNGなど)をアップロードします。
-   **その他**: 必要に応じて、プログラムのソースコードなどもアップロードできます。

#### ステップ3: 過去レポートの活用(オプション)

以前作成したレポートの構成や言い回しを再利用したい場合、過去のWordレポートを「テンプレートとして使用」エリアにアップロードできます。AIがそのレポートの構造を解析し、新しいレポートの基盤として活用します。

#### ステップ4: レポート生成の実行

必要なファイルをすべてアップロードしたら、「レポートを生成」ボタンをクリックします。バックグラウンドでAIが以下の処理を自動的に実行します。

-   データの統計処理
-   グラフの自動生成
-   理論値との比較分析
-   実験結果、まとめ、参考文献リストの文章生成

処理には数分かかる場合があります。進捗は画面に表示されますので、そのままお待ちください。

#### ステップ5: レポートの確認とダウンロード

生成が完了すると、プレビューが表示されます。内容を確認し、問題がなければ「Word形式でダウンロード」ボタンをクリックしてください。レポートが`.docx`形式のファイルとしてダウンロードされます。

**【重要】考察について**
本ツールは、皆様の学習機会を尊重するため、「考察」セクションの文章は自動生成しません。代わりに、考察のポイントやヒントを提示し、ご自身の言葉で記述するための「基盤」を提供します。AIが生成した客観的な分析結果を元に、深い考察を加えてレポートを完成させてください。

### 3.3. レポート履歴の管理

ダッシュボードの「レポート履歴」セクションでは、過去に作成したすべてのレポートが一覧表示されます。

-   **閲覧**: レポートのタイトルをクリックすると、内容をプレビューできます。
-   **再ダウンロード**: いつでも過去のレポートをWord形式で再ダウンロードできます。
-   **削除**: 不要になったレポートを削除できます。

## 4. よくある質問(FAQ)

**Q1: どのようなデータ形式に対応していますか？**
A1: Excel (.xlsx), 画像 (.jpg, .png), テキスト (.txt), ソースコード (.py, .cなど) に対応しています。詳細は随時アップデートされます。

**Q2: 考察も自動で書いてくれますか？**
A2: いいえ、考察は自動生成されません。本ツールは、客観的なデータ分析とレポートの骨子作成を支援し、学生の皆様が最も重要な「考察」に集中できるよう設計されています。AIによる分析結果を参考に、ご自身の考察を記述してください。

**Q3: 処理にはどのくらい時間がかかりますか？**
A3: アップロードされたデータの量や種類によって異なりますが、通常は2〜5分程度で完了します。

**Q4: セキュリティは安全ですか？**
A4: はい。アップロードされたデータや個人情報は、暗号化通信(HTTPS)で保護され、厳重に管理されます。ご本人の許可なくデータが外部に公開されることは一切ありません。
